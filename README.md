#  E-Commerce Sales Dashboard – Data Analytics Project (Excel)

##  Project Overview

I recently built an **Interactive E-Commerce Sales Dashboard** to analyze customer behavior, sales trends, and channel performance using **Microsoft Excel**.

This project focuses on generating meaningful business insights through data cleaning, pivot analysis, and interactive dashboard visualization.

---

##  Key Insights from the Analysis

-  Women contribute **~65% of total purchases**
-  Age group **30–49 years** generates **~50% of total revenue**
-  **Top performing states:**
  - Maharashtra  
  - Karnataka  
  - Uttar Pradesh  
-  Amazon is the highest contributing channel (**~35%**)
-  **92% order delivery rate** indicates strong operational performance
-  Seasonal sales peak observed during **mid-year**

##  Business Recommendation

To improve overall sales performance:

-  Target **women customers (30–49 years)**
-  Focus marketing campaigns in **top-performing states**
-  Run personalized offers and ads on:
  - Amazon  
  - Flipkart  
  - Myntra  

This strategy can help maximize revenue and customer engagement.
